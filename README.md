# Better Models

A modern, elegant browser for exploring AI models from multiple providers. Built with React, TypeScript, and Tailwind CSS.

## Features

- Browse AI models from various providers in a clean card-based interface
- Filter by provider, capabilities (Reasoning, Tool Call, Structured Output, etc.)
- Sort by name, release date, context size, or cost
- View detailed model information including pricing, token limits, and capabilities
- Copy model JSON or IDs with one click
- Dark/Light theme support
- Multi-language support (English, Chinese)
- Responsive design for all devices

## Tech Stack

- **Framework**: React 19 + TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS 4
- **UI Components**: Radix UI
- **Icons**: Lucide React
- **i18n**: i18next

## Getting Started

### Prerequisites

- Node.js 18+
- pnpm

### Installation

```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev

# Build for production
pnpm build

# Preview production build
pnpm preview
```
